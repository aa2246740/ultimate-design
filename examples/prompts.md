# Example Prompts

## Report Visualization

```text
Use $ultimate-design to turn this long research report into a visual executive page. Create a DESIGN.md contract from the prompt and source document, preserve the original user request, choose a non-generic visual direction, and verify the rendered page before delivery.
```

## Product UI

```text
Use $ultimate-design to design a dashboard for marketplace operators who need to triage supplier risk. Start from the product problem, define the success signal, create the minimum useful DESIGN.md, and implement a responsive prototype.
```

## Commercial Deck

```text
Use $ultimate-design to turn this memo into a 12-slide HTML PPT deck for a board meeting. Make the title-only reading tell the story, give every core slide one claim and evidence, and inspect every exported slide.
```

## Graphic / Social Carousel

```text
Use $ultimate-design to create 9 Xiaohongshu-style vertical cards explaining this product. Make the content concise, use a deliberate art direction, export all 9 PNGs, and provide a contact sheet.
```

## Audit And Repair

```text
Use $ultimate-design to critique this page and repair the highest-impact problems. Check requirement drift, content clarity, hierarchy, spacing, color fit, accessibility, responsive layout, and visual verification evidence.
```
