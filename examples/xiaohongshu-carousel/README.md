# Xiaohongshu Carousel Examples

Two 9-frame social carousel examples generated with Ultimate Design.

## Packages

- `zh/` - Chinese-context Xiaohongshu carousel. The copy was written for Chinese social reading, not translated from English.
- `en/` - English-context vertical carousel. The copy and layouts follow English maker/design launch conventions.

Each package includes:

- `index.html` - source HTML/CSS. Render one frame with `?id=1` through `?id=9`.
- `DESIGN.md` - request anchor, content model, taste signature, typography strategy, and export constraints.
- `png/` - exported 1080 x 1440 PNG frames.
- `contact-sheet.png` - visual overview for inspection.
- `verification-report.md` - render and validation notes.

These examples are social/screen drafts, not print-ready production files.
