# Ultimate Design Xiaohongshu English Carousel

Nine English-first vertical carousel frames promoting the Ultimate Design skill.

## Files

- `index.html` - source HTML/CSS for all 9 frames. Use `?id=1` through `?id=9` to render a single frame.
- `DESIGN.md` - design contract, content model, taste signature, and export constraints.
- `png/` - exported 1080 x 1440 PNG frames.
- `contact-sheet.png` - overview image for visual inspection.

## Format

- Platform target: Xiaohongshu-style vertical image carousel.
- Export size: 1080 x 1440 px.
- Ratio: 3:4.
- Copy language: English.
- Art direction: varied composition families, one shared palette strip, typography-led memory.

## Render

Use Chrome headless at 1080 x 1440 and capture `index.html?id=N` for each frame.
